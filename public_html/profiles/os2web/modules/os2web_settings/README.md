OS2Web Settings
==============================

Description
-----------
This module provides an configuration page on Admin. This includes
also other general settings for an OS2Web project.

Dependencies
-----------
- better_formats
- ctools
- customfilter
- features
- locale
- media
- media_youtube
- print
- strongarm
- pathauto
- workbench
- workbench_access
- workbench_files

Configuration
-----------
The configuration page is located at : admin/config/os2web

Installation
-----------
This module should reside in the modules directory of the installation,
most commonly profiles/os2web/modules/, but alternativly in sites/all/modules
(This could be for development purposes).

See https://github.com/syddjurs/os2web/wiki for further instructions.

This module can also be installed with drush make in your install profile.
